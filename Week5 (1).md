# 학습 내용 정리

## 39강. LOD

<!-- INCLUDE, EXCLUDE, FIXED 등 본 강의에서 알게 된 LOD 표현식에 대해 알게 된 점을 적어주세요. -->

LOD는 현재 시각화 수준과 관계없이 계산의 기준을 별도로 정의하는 기능이다

=> 데이터를 어떤 단위로 계산할지를 강제로 지정

LOD 종류

FIXED: 특정 차원 기준으로 고정
INCLUDE: 현재보다 더 세부적인 수준에서 계산
EXCLUDE: 현재보다 더 상위 수준에서 계산

## 40. LOD EXCLUDE

<!-- INCLUDE, EXCLUDE, FIXED 등 본 강의에서 알게 된 LOD 표현식에 대해 알게 된 점을 적고, 아래 두 질문에 답해보세요 :) -->

EXCLUDE는 현재 시각화에서 특정 차원을 제거하고 계산을 수행하는 방식이다

> **🧞‍♀️ FIXED와 EXCLUDE을 사용하는 경우의 차이가 무엇인가요?**

```
FIXED는 시각화와 관계없이 지정한 차원 기준으로 완전히 고정된 값을 계산
반면 EXCLUDE는 현재 뷰에서 특정 차원을 제거하여 계산
시각화 구조에 따라 결과가 달라진다
```

> **🧞‍♀️ 왜 ATTR 함수를 사용하나요?**

```
ATTR은 해당 값이 하나일 경우 그 값을 반환
여러 값이 존재하면 *을 반환한다

=>집계 수준이 맞지 않을 때 에러를 방지하고
단일 값인지 확인하기 위해 사용한다
```



## 41. LOD INCLUDE

<!-- INCLUDE, EXCLUDE, FIXED 등 본 강의에서 알게 된 LOD 표현식에 대해 알게 된 점을 적고, 아래 두 질문에 답해보세요 :) -->

INCLUDE는 현재보다 더 세부적인 수준에서 계산을 수행한 뒤 다시 집계하는 방식

> **🧞‍♀️ 그렇다면 어떤 경우에 각 표현식을 사용하나요? 예시와 함께 적어보아요**

```
FIXED: 특정 기준을 고정하고 싶을 때

예시
지역별 평균 매출 고정

INCLUDE: 더 세부 기준으로 계산이 필요할 때

예시
고객별 매출 평균

EXCLUDE: 특정 차원을 제거하고 싶을 때

예시
전체 평균과 비교
```



## 42. 매개변수

<!-- 매개변수에 대해 알게 된 점을 적어주세요 -->

사용자가 직접 값을 선택하여 분석 기준을 변경할 수 있도록 하는 기능

필터와 달리 데이터 자체를 거르는 것이 아니라 계산식이나 시각화 "기준"을 바꾸는 데 사용

주로 하나의 시트에서 여러 지표(매출/수익/수련)를 전환하거나, Top N과 같은 기준을 동적으로 변경할 때 활용

> **🧞‍♀️ 집합에도 매개변수를 적용할 수 있나요? 시도해봅시다**



## 43. 매개변수 실습

<!-- 영상 묶음에 포함되지 않아 찾기 어려우실까 링크를 아래에 첨부하겠습니다. 수강 후 삭제해주세요-->

매개변수는 단독으로 사용되지 않고 계산된 필드와 함께 활용된다

CASE 문을 이용하여 선택된 값에 따라 다른 측정값을 반환하도록 구성하면 하나의 시트에서 여러 지표를 전환할 수 있다

이 방식은 대시보드에서 반복되는 시트를 줄이고 사용자 인터랙션을 높이는 데 효과적임

## 44. 매개변수 실습

매개변수는 조건문과 결합하여 더 다양한 분석을 수행할 수 있다.

예를 들어 특정 조건에 따라 지표를 바꾸거나 비교 기준을 동적으로 설정하는 것이 가능

이를 통해 하나의 시각화 안에서 여러 분석 시나리오를 구현

## 46. 서식 계층

<!-- 서식계층에 대해 알게 된 점을 적어주세요 -->

마크카드는 시각화에서 데이터를 표현하는 방식을 결정하는 요소

색상, 크기, 레이블, 세부 정뽀, 툴립 등을 활용하여
하나의 그래프에 여러 정보를 동시에 표현할 수 있다.

데이터 자체가 아니라 “표현 방식”을 제어하는 핵심 기능


> **🧞‍♀️ 서식계층을 일반적인 것에서 구체적인 것 순서로 기입해보세요**


```
워크북 -> 워크시트 -> 축 -> 마크 -> 필드
하위 단계의 서식이 상위 설정을 덮어쓰기 때문에 원하는 결과를 얻기 위해서는 적용 위치를 정확히 이해해야 함
```

특히 강조해야 할 값이나 비교 대상이 명확하게 드러나도록 서식을 설정하는 것이 중요하다

2) 각 나라에서 가장 많이 주문한 사람의 이름을 첫 번째 열,

## 문제 2.

```
규영이는 태블로를 쓰실 수 없는 상사분께 보고하기 위한 대시보드를 만들고 싶어요. 

제품 중분류별로 구분하되 매개변수로써 수익, 매출, 수량을 입력하면 저절로 각각 지표에 해당하는 그래프로 바뀌도록 설계하고자 해요.

 어떤 값이 각 지표의 평균보다 낮은 값을 갖고 있다면 색깔을 주황색으로, 그것보다 높다면 파란색으로 표시하고 싶어요. 그 평균값은 각 지표별로 달라야 해요.
```
1. 매개변수를 생성하여 사용자가 지표를 선택할 수 있도록

2. 선택된 지표에 따라 값을 반환하는 계산된 필드를 생성

3.선택된 지표의 평균을 기준으로 값을 비교하여 색상을 구분

=> 이 방식은 하나의 시각화에서 여러 지표를 비교할 수 있도록 사용자 중심의 대시보드 구성을 가능하게 한다
