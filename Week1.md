# 1️⃣ 개념 정리

## 01. 여러 가지 과제 

## 빅데이터 환경에서 발생하는 역할 변화

-기업들이 빅데이터 분석 기반을 도입하면서 데이터는 하나의 플랫폼에 통합 저장됨

-데이터 활용을 위해 SQL을 이용한 데이터 추출 능력이 중요해짐

-분석 담당자와 엔지니어의 역할이 기존보다 확장됨

-Hadoop, Redshift, Hue, Aginity 등의 도구를 활용하여 SQL로 데이터를 추출하는 과정이 데이터 분석의 기본 단계

## 분석 담당자의 과제

1. 분석 지표를 스스로 정의해야 함

2. SQL을 이용하여 데이터를 직접 추출해야 함

3. SQL 없이 대용량 데이터 분석이 어려움

4. BI 도구만으로는 복잡한 분석 수행에 한계

5. 데이터 추출과 가공 과정으로 리포트 작성 시간 증가

6. SQL, 데이터 모델 등 새로운 기술 학습 필요

## 엔지니어의 과제

1. 분석 담당자의 요구 사항을 이해하기 어려움

2. 서비스 개발 SQL과 분석 SQL의 사용 방식 차이

3. 반복적인 데이터 요청 발생

4. 협업 과정에서 비효율 발생

5. 분석 목적에 맞는 데이터 제공이 어려움

## 분석 담당자 vs 엔지니어 갈등의 원인

사용 기술의 차이/ 업무 목표의 차이/ 데이터 이해 수준의 차이

->커뮤니케이션 문제와 협업 비효율 발생

해결 방향 = 역할 범위 확장

분석 담당자 = > SQL, 데이터 구조 이해 필요

엔지니어 =>  데이터 분석 구조 이해 필요

서로의 작업 영역 이해

데이터 구조 공유

협업을 통한 분석 환경 개선

빅데이터 시대에 필요한 능력

## 분석 담당자

SQL 활용 능력/ 데이터 가공 능력/ 데이터 모델 이해/ 분석용 데이터 추출/ 리포트 작성 능력

## 엔지니어

데이터 분석 구조 이해/ 집계 SQL 이해 / 데이터 활용 방식 이해/ 분석 요구 이해

## 02. 시스템

## PostgreSQL

-오픈소스 관계형 데이터베이스(RDBMS)
-표준 SQL을 잘 지원하며 다양한 확장 기능 제공
-테이블 구조로 데이터 저장
-인덱스를 이용한 빠른 데이터 조회 가능
-대규모 데이터 처리에는 단일 서버 구조로 한계 존재

## 특징

-관계형 데이터베이스 구조
-표준 SQL 지원
-설치 및 사용이 쉬움
-대용량 데이터 처리에는 한계

## Apache Hive

Hadoop 기반의 분산 데이터 처리 시스템
HDFS에 저장된 데이터를 SQL 형태로 조회 가능
MapReduce 기반으로 대량 데이터 처리
실행 속도가 느려 실시간 분석에는 부적합

## 특징

분산 파일 시스템 기반
대용량 데이터 처리 가능
SQL 인터페이스 제공
배치 처리에 적합

## Amazon Redshift

AWS에서 제공하는 클라우드 기반 데이터 웨어하우스
분산 데이터베이스 구조 사용
컬럼 기반 저장 방식으로 빠른 조회 가능
클라우드 환경에서 확장 가능

## 특징

분산형 RDB
컬럼 기반 저장
빠른 분석 속도
사용량 기반 비용 발생

## Google BigQuery

구글에서 제공하는 서버리스 데이터 분석 시스템
SQL을 이용하여 대규모 데이터 분석 가능
서버 관리 없이 사용 가능
조회한 데이터 양에 따라 비용 발생

## 특징

서버 관리 불필요
사용량 기반 과금
대용량 데이터 처리 가
SQL 지원

## SparkSQL

Apache Spark 기반 데이터 처리 시스템
SQL을 이용하여 분산 데이터 처리 가능
Python, Java, Scala 등 다양한 언어 지원
대용량 데이터 분석 및 머신러닝 작업 가능

## 특징

분산 처리 프레임워크
다양한 언어 지원
빠른 데이터 처리
복잡한 데이터 분석 가능

## 데이터의 종류

데이터 분석에서 사용하는 데이터는 크게 업무 데이터와 로그 데이터로 구분됨

## 업무 데이터
서비스 운영 과정에서 데이터베이스에 저장되는 데이터
정확성이 중요하며 트랜잭션 중심 데이터

## 트랜잭션 데이터
사용자의 행동으로 발생하는 데이터

구매 기록/ 결제 기록/ 주문 데이터

## 마스터 데이터
서비스 운영에 필요한 기준 데이터

사용자 정보/ 상품 정보/ 카테고리 정보

## 로그 데이터
사용자의 행동이나 시스템 상태를 기록한 데이터

페이지 방문 기록/ 클릭 기록/ URL
