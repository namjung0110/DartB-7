## 20강: 파이와 도넛차트

파이 차트는 전체 대비 비율을 시각적으로 표현하는 차트이며, 도넛 차트는 이를 응용한 형태

<도넛 차트 생성 방법>

1.세그먼트와 매출을 추가한 후 차트 유형을 파이 차트로 변경
2.열 선반에 0을 입력하여 임의의 축 생성
3.해당 축을 복제하여 두 개의 파이 차트 생성
4.두 번째 파이 차트를 흰색으로 설정하고 크기를 줄임
5.이중축 적용 후 축과 격자 제거하여 도넛 형태 완성

## 21강: 워드와 버블차트

버블 차트는 수치 데이터를 원의 크기로 표현하는 시각화 방식
=> 값이 클수록 원이 크게

워드 클라우드는 텍스트 데이터에서 단어의 등장 빈도를 크기로 표현하는 차트

버블차트 특징 :
1.원의 크기로 표현하여 직관적인 비교 가능
2.색상을 활용하면 추가 변수까지 함께 표현 가능

워드 클라우드 특징
1.단어의 등장 빈도를 기반으로 크기 표현
2.색상으로 추가적인 정보 표현 가능

단점 
버블 차트: 정확한 수치 비교보다는 상대적인 크기 비교에 적합함
워드 클라우드: 단어 간 문맥을 고려하지 못하는 한계가 있음

## 22강: 이중축과 결합축

이중축 
하나의 뷰에서 두 개의 축을 각각 독립적으로 사용하는 방식

특징

1.서로 다른 단위의 데이터를 동시에 표현 가능
2.각 축에 서로 다른 그래프 형태 적용 가능

결합축 
여러 측정값을 하나의 축에서 함께 표현하는 방식

특징 

1.여러 측정값을 하나의 축에서 비교
2.측정값 선반을 활용하여 여러 값 동시에 표현

단점
이중축 :  사용 시 축을 동기화하지 않으면 해석 오류 발생 가능
결합축 :  단위가 같은 경우에만 사용하는 것이 적절함

## 23강: 분산형 차트

분산형 차트 
두 변수 간의 관계를 시각적으로 표현하는 차트로 각 데이터를 점으로 나타내어 상관관계를 분석

특징
1.매출과 수익처럼 두 변수 간 관계 파악 가능
2.색상이나 세부 정보를 활용해 그룹별 분석 가능

주사용용도

추세선을 추가하여 전체적인 흐름 파악
이상치(outlier)를 확인하여 특이한 데이터 발견

<img width="581" height="303" alt="image" src="https://github.com/user-attachments/assets/b87c5dac-7507-4d8a-b566-93be6c049c1f" />

```js
강의 영상과 달리, 우리 파일에는 '제조 업체' 필드가 없습니다. 필요한 경우, 계산된 필드를 이용해 'SPLIT([제품 이름], ' ', 1)'를 '제조 업체'로 정의하시고 세부 정보에 놓아주세요.
```



## 24강: 히스토그램
히스토그램 
연속형 데이터를 일정한 구간으로 나누어 각 구간에 속하는 데이터의 빈도를 시각적으로 표현하는 차트
데이터의 분포 형태를 파악하는 데 매우 유용하며, 값이 어디에 몰려 있는지 확인할 수 있다.

특징
연속형 데이터를 구간 단위로 나누어 표현
막대 간 간격이 없음 
데이터의 분포, 집중 구간, 퍼짐 정도를 파악 가능

## 25강: 박스플롯

박스플롯은 데이터의 분포와 이상치를 동시에 확인할 수 있는 차트로, 다섯 가지 요약값을 기반으로 구성된다.

박스플롯 구성 요소
최소값 / 1사분위(Q1) / 중앙값(Q2) / 3사분위(Q3) / 최대값

IQR(= Q3 - Q1)을 기준으로 이상치 판단

데이터의 퍼짐 정도와 중앙값을 동시에 파악 가능

## 26강: 영역차트

<img width="466" height="311" alt="image" src="https://github.com/user-attachments/assets/d7aa8c01-ad4a-401f-9202-e527046dde88" />

영역차트는 라인 차트 아래 영역을 색으로 채운 형태로, 시간에 따른 데이터의 변화와 누적 흐름을 동시에 표현

특징
시간 흐름에 따른 변화 확인 가능
누적값을 시각적으로 표현
구성 비율 변화와 추이를 함께 분석 가능

활용 방법
세그먼트를 색상으로 구분하여 누적 영역 표현
구성 비율(%) 형태로 변환하여 비중 분석 가능

## 27강: 간트차트

간트차트는 시간에 따른 작업 기간을 시각적으로 표현하는 차트로, 프로젝트 일정 관리에 주로 사용된다.

생성 방뻡
DATEDIFF 함수를 활용하여 기간 계산
계산된 기간을 크기(Size)에 배치
평균 집계로 설정하여 왜곡 방지
시작 시점과 지속 기간을 동시에 표현 가능

활용 방법
배송 기간, 프로젝트 일정 등 시간 기반 분석
카테고리나 고객 기준으로 세분화 가능


## 28강: 필터
필터는 데이터 분석에서 특정 조건에 맞는 데이터만 선택하여 시각화하는 기능으로, 결과뿐 아니라 성능에도 영향
적용 순서
추출 -> 데이터 원본 -> 컨텍스트 -> 차원 -> 측정값
필터 적용 순서에 따라 결과가 달라질 수 있음

컨텍스트 필터 특징
다른 필터보다 먼저 적용됨
이후 필터들이 해당 결과를 기준으로 작동

## 29강: 그룹
그룹은 데이터 원본에 없는 사용자 정의 범주를 만들어 여러 항목을 하나로 묶는 기능
생성 방법
차트에서 항목 선택 후 그룹 생성
필드에서 직접 그룹 생성

->데이터를 원하는 기준으로 재구성 가능

활용 방법
 데이터 단순화 및 해석 용이
 그룹을 활용하면 복잡한 데이터를 더 직관적으로 표현 가능
