# 1️⃣ 학습 내용 정리

## 10강: 차원과 측정값

<!-- 차원과 측정값에 관해 배우게 된 점을 적어주세요 -->

> **🧞‍♀️ 차원과 측정값의 고유한 특성에 대해 설명해주세요.**

```
차원(Dimension)은 데이터를 구분하거나 범주화하는 역할을 하는 필드이다.
주로 문자열이나 날짜와 같은 정성적 정보를 포함하며 데이터를 분류하는 기준이 된다.
예를 들어 국가, 지역, 카테고리, 고객 이름 등이 차원에 해당한다.
측정값(Measure)은 수치 데이터를 의미하며 집계가 가능한 값이다.
매출, 수익, 판매 수량 등과 같이 합계, 평균, 최대값 등의 계산이 가능하다.
즉 차원은 데이터를 분류하는 기준이고,
측정값은 계산이나 집계의 대상이 되는 값이라고 볼 수 있다.
```

*Quiz: 차원은 불연속형이고, 측정값은 연속형이다 (o, x)*

정답: o



## 11강: 시각화

<!-- 시각화 관해 배우게 된 점을 적어주세요 -->

태블로에서 시각화는 데이터 필드를 시트에 배치하여 그래프나 표로 표현하는 과정이다.
차원과 측정값을 어떻게 배치하느냐에 따라 다양한 시각화 형태가 만들어진다.

예를 들어

차원 → 축 또는 구분 기준

측정값 → 그래프의 크기나 값

으로 사용된다.

또한 드래그 앤 드롭 방식으로 쉽게 시각화를 만들 수 있으며
여러 필드를 동시에 선택하면 다양한 그래프를 빠르게 생성할 수 있다.

*Quiz: 차원은 대부분 불연속형이며 표 형태로 시각화했을 때 머리글로 표시되고, 측정값은 대부분 연속형이므로 표 형태로 시각화 진행했을 때 패널로 추가된다.(o, x)*

정답 : 0

## 12강: 막대그래프

<!-- 막대그래프에 관해 배우게 된 점을 적어주세요 -->

막대그래프는 범주별 값을 비교할 때 가장 기본적으로 사용하는 시각화 방법이다.

특징

-여러 범주 간의 크기 비교가 쉽다
-데이터 차이를 직관적으로 표현할 수 있다
-정렬을 통해 순위 비교도 가능하다
ex)
카테고리별 매출
지역별 판매량
상품별 수익
같은 데이터를 분석할 때 막대그래프를 사용하면 효과적으로 비교할 수 있다.

## 13강: 누적막대그래프

<!-- 누적막대그래프에 관해 배우게 된 점을 적어주세요 -->
누적막대그래프는 전체 값에서 각 범주가 차지하는 비중을 표현할 때 사용하는 그래프이다.
특징
-전체 대비 구성비 확인 가능
-여러 범주의 합계를 한 번에 확인 가능
-시간에 따른 구성비 변화 분석 가능
ex)
카테고리별 매출 비율
지역별 판매 비중
제품군별 매출 구성

테이블 계산을 활용하여 백분율 형태로 표현하면
각 항목이 전체에서 차지하는 비율을 쉽게 확인할 수 있다.

## 14강: 병렬막대그래프

<!-- 병렬막대그래프에 관해 배우게 된 점을 적어주세요 -->
병렬막대그래프는 여러 범주를 동시에 비교하기 위해 막대를 나란히 배치하는 그래프이다.
![alt text](https://raw.githubusercontent.com/DArt-B-Official/07_Tableau_Template/main/images/Week1-1.png)
날짜 데이터의 경우 태블로가 시간 흐름을 인식하도록
불연속형을 연속형으로 변환하면 시간 흐름에 따른 비교가 가능하다


> *🧞‍♀️ 끊어진 색상으로 배치되어 표현되는 경우와 이어지는 그라데이션 색으로 표현되는 경우 두 가지가 있습니다. 위 사진의 경우 왜 색깔이 끊어지는 색상으로 표시되지 않고 그라데이션으로 표시되었나요? 데이터의 특성과 관련하여 이야기해 봅시다.*

```
해당 데이터가 연속형 데이터이기 때문이다.
태블로에서는
불연속형 데이터 → 범주형 색상
연속형 데이터 → 값의 크기에 따른 그라데이션 색상
으로 표현된다.
따라서 값의 크기를 기준으로 색상이 점진적으로 변화하는
그라데이션 색상이 적용된 것이다.
```

## 문제1

```js
혜인이는 superstore의 한국 수출 관리 업무를 맡고 있습니다. 국가/지역이 우리나라, 즉 'South Korea'인 데이터만을 필터링하여, 상품 하위범주 별로 각 하위범주가 매출의 비율 중 얼마만큼을 차지하는지를 트리맵으로 확인하고 싶습니다. 트리맵의 각 네모 안에 표시되는 텍스트에는 **해당 범주의 이름과 전체에서 해당 범주가 차지하는 비율**이 표시되었으면 합니다.

혜인이를 도와주세요! (풀이를 찾아가는 과정을 기술해주세요)
```

```
1. Sample Superstore 데이터를 불러온다.
2. Country/Region 필드를 필터로 드래그하여
South Korea만 선택한다.
3. Sub-Category를 마크 카드의 Label에 배치
Sales를 Size에 배치
4. Sales 필드에서
Quick Table Calculation → Percent of Total 선택
5. Sub-Category 이름과 매출 비율이 표시되도록
Label에 추가한다.
6. Sales를 Color에 추가하여
매출 규모에 따라 색상 차이를 표현한다.
```
